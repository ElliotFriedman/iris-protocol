# Iris Protocol — Logo & Iconography

Apply when placing the logo or choosing icons.

## The Iris Mark

A geometric iris aperture — six overlapping blades forming a hexagonal center opening. Thin, precise strokes in Iris purple (`#7B2FBE`) on dark backgrounds, Void (`#0D0D14`) on light surfaces.

**Construction:** Six blades at 60° intervals. Curved trapezoid blades with rounded inner edge. Hexagonal aperture. Stroke: 2px at 48px size.

## Logo Variants

| Variant | Usage |
|---------|-------|
| Mark only | Favicon, small icons, social avatars |
| Mark + wordmark (horizontal) | README header, pitch deck title, nav |
| Wordmark only | Text-only contexts, footer, code comments |

**Wordmark:** "Iris" in JetBrains Mono Bold, `#7B2FBE`. "Protocol" in JetBrains Mono Regular, `#8A8A9A`. Always lowercase: `iris protocol`.

## Logo Don'ts

- Never rotate, stretch, or distort
- Never change blade count
- Never add shadows, glows, or 3D effects
- Never place on low-contrast backgrounds
- Never use below 24px (mark only below 32px)

## Iconography

Use **Lucide** icons (lucide.dev). Stroke: 1.5px. Size: 20px default, 16px dense, 24px nav. Color: Ash inactive, Bone active, accent for semantic.
