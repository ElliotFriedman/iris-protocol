# Iris Protocol — Slide Design

Apply when creating pitch deck or presentation slides.

## Foundations

- **Background:** Void (`#0D0D14`), full bleed. No gradients.
- **Title:** JetBrains Mono Bold, Bone (`#E8E6E1`), left-aligned. Max 8 words.
- **Body:** Satoshi Medium, Ash (`#8A8A9A`), left-aligned. Max 3–4 lines.
- **Accent:** Thin horizontal rules (1px, Graphite). Iris purple for one key element per slide only.
- **Grid:** 12-column, 40px margins. Content in columns 2–11.

## Code on Slides

JetBrains Mono Regular on Onyx (`#22223A`) with 1px Graphite border. Syntax: Cyan for types, Iris for keywords, Mint for strings.

## Data/Metrics

JetBrains Mono Bold, Cyan (`#00F0FF`), 48px+. Caption below in Satoshi Regular, Ash.

## Layout Patterns

- **Type A — Statement:** One sentence, 28–36px JetBrains Mono Bold, centered vertically, left-aligned.
- **Type B — Diagram:** Title top-left, diagram 70% of area, caption bottom-left.
- **Type C — Comparison:** Split left/right with thin Graphite vertical divider.
- **Type D — Demo:** Dark screenshot with Iris purple circle annotations and Satoshi captions.
