# Iris Protocol — Social Media

Apply when creating social posts, profile assets, or tweet visuals.

## Twitter/X Profile

- **Avatar:** Iris mark on Void (`#0D0D14`) background
- **Banner:** Void background, wordmark left-aligned, tagline right-aligned in Ash
- **Bio:** "Trustless embedded wallets for AI agents. Every permission onchain."

## Tweet Visuals

- Code screenshots: dark terminal theme matching brand palette
- Diagrams: follow `/brand-diagrams` rules, cropped to 16:9
- Quote cards: Void background, JetBrains Mono Bold in Bone, 4px Iris purple left border
- Never use stock images, generic crypto graphics, or AI-generated imagery

## Thread Structure

1. Hook tweet (max 200 chars, no hashtags) — the one-sentence value proposition
2. Context — what problem exists, one tweet
3. Mechanism — how Iris solves it, 1–2 tweets
4. Implication — what this enables, one tweet
5. CTA or link — where to learn more

## Reply Register

Match the energy of the incoming message but never drop below Precise + Understated. For criticism: acknowledge the technical point, respond with facts. For praise: brief thanks, redirect to the work. Never use exclamation marks in any brand copy (threads, replies, bios — all surfaces).

Threads use Narrative mode (see `/brand-voice`). Claims like "first" or "only" are permitted but must be falsifiable and specific.

## README Header

Minimal: iris mark (32px) + "iris protocol" in JetBrains Mono + primary tagline as blockquote. No badges, no shields, no decorative separators.
